1. Windows

<hr/>

### Supported Features

- Shortcuts for basic commands
  - ctrl + c : copy array
  - ctrl + v : past array (next to the currently active array)
  - ctrl + n : new file
  - ctrl + o : open file
  - ctrl + s : save file
  - ctrl + shift + s : save file as
- Drag to copy current cell value
- Toggle to view cell values
- Scroll to change cell values
